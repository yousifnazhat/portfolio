# Blueprint

## Overview

This is a portfolio website designed to showcase projects and skills. It features a modern, single-page layout with smooth scrolling navigation.

## Design and Features

### General
- Single-page layout
- Smooth scrolling navigation
- Responsive design for mobile and web

### Homepage
- Hero section with a brief introduction and links to resume.
- About Me section.
- Tech stack ticker.
- Interactive project cards for software and hardware projects.

### Project Pages
- Dedicated pages for each software project:
    - RISC-V Simulator
    - Election Engine
    - Unix FS Emulator
    - 128-Bit Toolkit
    - Symbiote Host Compatibility Tree
- Each project page includes:
    - A detailed description of the project.
    - A link to the GitHub repository.
    - A project diagram.
    - A back button to return to the portfolio.
